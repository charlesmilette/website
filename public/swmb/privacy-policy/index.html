SearchWithMyBrowser privacy policy

Your use of this application consists in an agreement of this policy.

Content of the privacy policy

This application doesn't:
  • Collects & uses any personal information
  • Stores any personal data
  • Shares it to third parties

All data required for operation, including but not limited to:
  • Cortana online search queries
  • Internet addresses opened using the "Continue with PC" feature of Windows
is treated for proper function of the application and then immediatly dropped (never stored or shared).

Normal Windows telemetry features might send such information to the developer, which will be used only for debugging and improvement of the application and will never be shared with third-parties.
